1. Cómo funciona1. How it works

  1. Escriba fórmulas, texto y bloques de código en el panel Code (izquierda).Write formulas, text and code blocks in the Code panel (left).
  2. Presione F5 o haga clic en ▶ Calculate para evaluar.Press F5 or click ▶ Calculate to evaluate.
  3. Los resultados aparecen en el panel Output (derecha).Results appear in the Output panel (right).
  4. Use Insert para insertar bloques, funciones y comandos CAD.Use Insert to insert blocks, functions and CAD commands.
AutoRun: Si está activado, el output se actualiza automáticamente al escribir. AutoRun: When enabled, output updates automatically as you type.
